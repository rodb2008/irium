## 📤 Upload Methods

### Method 1: GitHub Web Interface (Recommended)
1. Go to https://github.com/iriumlabs/irium
2. Navigate to the main branch
3. Click "Add file" → "Upload files"
4. Drag and drop all files from the archive
5. Commit with message: "Add wallet integration and node/miner scripts"
6. Push to main branch

### Method 2: Download Archive
1. Download: `irium-wallet-integration-complete.tar.gz` (766KB)
2. Extract on your local machine
3. Upload files via GitHub web interface or GitHub Desktop

### Method 3: Manual File Upload
Use the file list in `files-to-upload.txt` to upload each file individually.

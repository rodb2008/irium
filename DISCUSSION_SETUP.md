# GitHub Discussions Setup Guide

## Recommended Categories

### 📢 Announcements
- **Description**: Official announcements from the Irium team
- **Format**: Announcement (only maintainers can post)
- **Use for**: Release announcements, network updates, important news

### 💡 Ideas
- **Description**: Share ideas for improving Irium
- **Format**: Open discussion
- **Use for**: Feature requests, protocol improvements, ecosystem ideas

### 🙋 Q&A
- **Description**: Ask questions about Irium
- **Format**: Question/Answer
- **Use for**: Technical questions, getting started help, troubleshooting

### 🛠️ Development
- **Description**: Technical discussions about Irium development
- **Format**: Open discussion
- **Use for**: Code reviews, architecture discussions, development roadmap

### ⛏️ Mining
- **Description**: Everything about mining IRM
- **Format**: Open discussion
- **Use for**: Mining setup, pool discussions, hardware recommendations

### 💰 Wallets & Transactions
- **Description**: Wallet and transaction discussions
- **Format**: Open discussion
- **Use for**: Wallet setup, transaction issues, address questions

### 🌐 Network & Nodes
- **Description**: Running nodes and network infrastructure
- **Format**: Open discussion
- **Use for**: Node setup, P2P issues, network health

### 📊 General
- **Description**: General Irium discussions
- **Format**: Open discussion
- **Use for**: Community chat, introductions, off-topic

# 🎉 Welcome to the Irium Community!

Welcome to the official Irium blockchain community discussions!

## About Irium

Irium is a SHA-256d Proof-of-Work blockchain with 8 unique innovations:
- Zero-DNS Bootstrap
- Self-healing Peer Discovery
- Genesis Vesting with On-chain CLTV
- Per-Transaction Relay Rewards
- Sybil-resistant P2P Handshake
- Anchor-File Consensus
- Light Client Support (SPV/NiPoPoW)
- On-chain Metadata Commitments

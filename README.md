# Irium Blockchain (IRM)
A next-generation proof-of-work blockchain designed for true decentralization.

- Latest Release: v1.0.0 – Production (stable mining + P2P sync)
  https://github.com/iriumlabs/irium/releases/tag/v1.0.0

## Why Irium?
- Zero-DNS Bootstrap
- Ultra-Low Fees (0.0001 IRM)
- Fair Launch (3.5% timelocked vesting, no premine/ICO)
- SHA-256d (Bitcoin-compatible)
- SPV + NiPoPoW
- Relay rewards

## Specs (short)
Algo: SHA-256d | Max: 100M IRM | Mineable: 96.5M IRM
Block time: ~13m | Halving: 210k | Retarget: 2016 | Maturity: 100
Min fee: 0.0001 IRM | P2P: 38291

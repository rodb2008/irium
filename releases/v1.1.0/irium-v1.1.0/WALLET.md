# Irium Wallet Guide

## Create Wallet
```bash
python3 scripts/irium-wallet-proper.py create
```

## Backup Wallet
File location: ~/.irium/irium-wallet.json

IMPORTANT: Back up this file securely!

## Send IRM
```bash
python3 scripts/irium-wallet-proper.py send <address> <amount>
```

## Check Balance
```bash
python3 scripts/irium-wallet-proper.py balance
```

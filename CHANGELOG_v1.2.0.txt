Irium v1.2.0 - P2P Network Resilience & Stability (October 24, 2025)

CRITICAL FIXES:
=============

1. Runtime Seedlist Registration (Network Resilience)
   - Added peer registration for BOTH incoming AND outgoing connections
   - VPS and all nodes now save discovered peers to seedlist.runtime
   - Network can survive VPS/node restarts without losing peer information
   - Peer database tracks connection history

2. Duplicate Connection Prevention (IP:PORT Deduplication)
   - Fixed: Now checks exact IP:PORT combination, not just IP
   - Allows multiple services from same IP (e.g., node:38291 + miner:38292)
   - Prevents duplicate connections to same service
   - Improves connection stability

3. Block Relay Spam Fix (Performance)
   - Implemented relay tracking to prevent duplicate block broadcasts
   - Each block only relayed ONCE per peer
   - Eliminates network spam (was relaying hundreds of times)
   - Reduces bandwidth usage significantly

4. Self-Connection Detection (Public IP Support)
   - Detects public IP at startup to prevent self-connection loops
   - Supports NAT environments properly
   - Checks localhost, public IP, and local IP
   - Eliminates connection timeout errors for own IP

5. Peer Height Tracking
   - Peer heights update after receiving blocks
   - Improves sync accuracy and peer selection
   - Better network state awareness

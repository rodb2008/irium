# Pilot Tester Self-Serve Guide

1. Open the pilot UI (`/`) and create a session.
2. Keep your `swap_id` and `session_token` private.
3. Fund only the `btc_htlc_address` shown in your session.
4. Track progress on the status page until terminal state.
5. If status stalls, share `swap_id`, txids, and timestamp with operators.

Warnings:
- Testnet/trial only.
- Do not use real funds.
- Session token is required for your status/events view.

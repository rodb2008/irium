# Irium API Reference

## Explorer API (Port 8082)

### GET /api/stats
Get blockchain statistics

### GET /api/latest?count=10
Get latest blocks

### GET /api/block/{height}
Get block by height

## Wallet API (Port 8080)

### GET /api/wallet/status
Get wallet status

### GET /api/wallet/balance
Get wallet balance

### GET /api/network/info
Get network information
